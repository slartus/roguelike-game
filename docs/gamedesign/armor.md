# Броня

**Статус:** не реализована. Файл — placeholder + дизайн-заготовка.

## Текущее состояние

- Игрок принимает урон 1:1 (`Player.take_damage(amount)` — вычитает `amount` из `health`).
- Никаких модификаторов, резистов, брони, щитов нет.

## Планируемый дизайн (не имплементировано)

Идея: параллельно `WeaponResource` завести `ArmorResource`, `Player` держит `equipped_armor`, урон проходит через модификаторы.

Возможные поля `ArmorResource`:

| Поле | Тип | Смысл |
|------|-----|-------|
| `display_name` | `String` | Название |
| `damage_reduction` | `int` | Флат-снижение урона (min 0 после вычета) |
| `damage_reduction_pct` | `float` | Процентное снижение (0.0–0.9) |
| `move_speed_mult` | `float` | Множитель скорости (>1 = легкая, <1 = тяжёлая) |
| `color` | `Color` | Тонирование игрока |

Порядок применения при `take_damage(amount)`:
1. `amount *= (1.0 - damage_reduction_pct)`
2. `amount = max(0, amount - damage_reduction)`
3. `health -= amount`

Пикап через `ArmorPickup` (аналогично `WeaponPickup`), спавнится из отдельного `ArmorChest` или расширения существующего chest'а.
