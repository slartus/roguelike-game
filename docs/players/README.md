# Гайд игрока

Всё что полезно знать, чтобы прожить дольше пары комнат.

- [Бестиарий](bestiary.md) — кто на тебя охотится, HP и повадки.
- [Оружие](weapons.md) — сравнение того, что можно поднять с пола.
- [Эффекты](effects.md) — статусы, которые могут висеть на тебе (яд, паутина).

Для деталей реализации (формулы, тайминги, поведение AI) — см. [docs/gamedesign](../gamedesign/README.md).
