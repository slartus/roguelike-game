# Комнаты

## Общее

Все комнаты — арены 480×270 пикселей (базовый viewport). По периметру — 4 `StaticBody2D`-стены толщиной 8 px. В центре нижней стены — точка перехода в следующую комнату (`Door`).

## Обязательный интерфейс room-сцены

Любая новая комната обязана содержать эти ноды:

| Нода | Тип | Назначение |
|------|-----|------------|
| `SpawnPoints` | `Node2D` | Родитель для 6 `Marker2D` (позиции спавна врагов) |
| `SpawnPoints/Spawn1..6` | `Marker2D` | Точки, откуда `Main` рандомно берёт 3–6 для спавна врагов |
| `PlayerStart` | `Marker2D` | Стартовая позиция игрока при загрузке комнаты |
| `Door` | инстанс `door.tscn` | Появляется когда все враги мертвы; при контакте с игроком → следующая комната |

`Main` работает с комнатой через этот интерфейс — не полагаться на внутренние ноды конкретной комнаты.

## Варианты планировок

| Файл | Планировка | PlayerStart |
|------|-----------|-------------|
| `scenes/rooms/room.tscn` | Пустая арена | (240, 200) |
| `scenes/rooms/room_pillars.tscn` | 4 столба 16×16 в позициях (120, 90), (360, 90), (120, 180), (360, 180) | (240, 220) |
| `scenes/rooms/room_cross.tscn` | Крест в центре: горизонтальная стена 120×8 + вертикальная 8×100 в (240, 135) | (240, 220) |

`Main` при `_ready()` случайно выбирает одну из 3 через `pick_random()`.

## Дверь (`door.tscn`)

- `Area2D` 32×16 в центре нижней стены (позиция `Door` в комнате: `(240, 262)`).
- При спавне скрыта (`visible = false`, `monitoring = false`).
- Метод `open()` включает видимость и мониторинг.
- Сигнал `player_entered` эмитится при `body_entered` от ноды в группе `player`.

## Boss-комнаты

Специальный режим `Main`, а не отдельная сцена комнаты:

- Триггер: `GameState.current_room_number % 5 == 0` (см. `BOSS_ROOM_INTERVAL` в `main.gd`).
- Обычные враги на `SpawnPoints` **не спавнятся**.
- Один босс спавнится в позиции `(240, 110)` (`BOSS_POSITION`).
- Chest в boss-комнатах **не появляется** — он смещён на следующую обычную комнату по правилу interval'а.

Планировка boss-комнаты — та же случайная из 3 вариантов; специальной boss-арены нет.
